# brick.js

A HTML Canvas breakout game inspired by the iPod Nano game [Brick](https://apple.fandom.com/wiki/Games_on_1st_and_2nd_Gen._iPod_Nano#Brick).

## Development
### Install dependencies
```sh
pnpm install
```
### To run dev server
```sh
pnpm dev
```

### Run unit tests
```sh
pnpm test
```

## Build
### To build
```sh
pnpm build
```
### To serve build locally
```sh
pnpm preview
```
